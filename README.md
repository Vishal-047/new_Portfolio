# Dev Personal Portfolio

A sleek, responsive personal portfolio built with React and Vite.

## 🚀 Features

- **Modern Tech Stack**: Built with React 19, Vite, and Tailwind CSS v4.
- **Fast Performance**: Powered by modern build tools for lightning-fast load times.
- **Responsive Design**: Fully mobile-friendly UI using Tailwind's utility classes.
- **Interactive UI**: Enhanced user experience with `lucide-react` icons.
- **Contact Form**: Integrated with `@emailjs/browser` for seamless communication.

## 🛠️ Getting Started

### Prerequisites

Ensure you have [Node.js](https://nodejs.org/) installed on your system.

### Installation

1. Clone the repository and navigate into the project directory.
2. Install dependencies:
   ```bash
   npm install
   ```

### Development Server

To start the development server:

```bash
npm run dev
```

Open your browser and navigate to `http://localhost:5173` to view the application in action.

### Building for Production

To create a production-ready optimized build:

```bash
npm run build
```

This will generate a `dist` folder with the compiled and minified assets.

To preview the production build locally:

```bash
npm run preview
```

## 📦 Key Dependencies

- `react` & `react-dom`
- `tailwindcss`
- `@tailwindcss/vite`
- `lucide-react`
- `@emailjs/browser`
